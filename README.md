
# 📊 Superstore Sales Analysis | End-to-End Data Project

This project demonstrates a complete Data Analytics workflow, starting from raw data processing to creating an interactive business dashboard. It uses a combination of **Python**, **SQL**, and **Power BI** to extract meaningful insights from a Superstore dataset.

## 🚀 Project Overview
The main goal of this project was to analyze sales performance, identify profitable customer segments, and visualize regional trends to help in data-driven decision-making.

## 🛠️ Tech Stack Used
* **Data Cleaning & EDA:** Python (Pandas)
* **Database Management:** SQL (Integrated Queries)
* **Visualization:** Power BI Desktop
* **Environment:** Google Colab

## 📂 Key Steps in the Project
1. **Data Preprocessing (Python):** Handled missing values, standardized date formats, and performed data type conversions using Pandas.
2. **Advanced Analysis (SQL):** Used SQL queries to calculate Total Revenue, Average Discount, and segment-wise Order counts.
3. **Interactive Dashboarding (Power BI):** Developed a dynamic dashboard to visualize:
   * Sales and Profit Trends over time.
   * Regional Sales Performance.
   * Category and Sub-category wise distribution.
